# Documents sample map

Use this file when the target path is a `Documents`-style root or one of its immediate subfolders.

Sample buckets:
- `Admin`
- `Capture`
- `Courses`
- `Personal`
- `Projects`

Suggested routing:
- `Admin`
  - official forms
  - invoices
  - reimbursement records
  - tax or visa documents

- `Capture`
  - screenshots
  - screen recordings
  - exported meeting captures
  - temporary media exports

- `Courses`
  - keep each course folder as a whole by default
  - do not restructure course internals unless the user explicitly targets one course

- `Personal`
  - CV or resume files
  - profile photos
  - biography notes
  - personal reference documents

- `Projects`
  - active project folders
  - research notes
  - writing drafts
  - review packages

Naming rule:
- for organizational folders created by this workflow, capitalize the first word only

This map is a sample, not a required layout.
